## 🏢 公司簡介

**公司名稱**：順星智能科技有限公司（Mas Intelligent Technology Ltd.）  
**性質**：台灣新創公司，專注於金融資訊軟體服務

---

## 📦 主要產品與服務核心

一個整合 AI 與多券商 API 的**程式交易平台**，協助投資人（尤其是非程式背景者）更輕鬆地進行自動化交易，提供以下解決方案與優勢：

### ✅ 市場痛點

手動交易不穩定，易受情緒與時間限制  
程式交易門檻高，缺乏標準 API，策略難以驗證與部署  
投資人轉向低手續費券商時面臨高技術與整合成本

### 🌟 平台特色優勢

**AI 輔助交易助理**：語音、圖像、文字輸入產生個人化策略與程式碼  
**整合多家主流券商 API**：一站式接入與切換  
**模擬 / 真實交易無縫切換**：策略驗證後一鍵部署  
**策略回測與優化功能**：提升策略獲利與風控能力  
**量化分析社群 + MAS Coin 生態**：高手策略交易與新手交流共享

---

## 💻 平台操作流程與技術整合

### 🧠 一般使用者（AI 助理用戶）操作流程

1. 從平台介面選擇或輸入自訂或非自訂交易邏輯  
2. 自動產生可執行的程式交易策略程式碼  
3. 系統於後端進行回測，產出績效報表  
4. 顯示靜態 / 動態 KPI 報表給使用者檢視  
5. 儲存策略後可**下載真實交易執行檔**  
6. 使用者只需輸入券商私服器、帳號、密碼，即可執行實際交易

### 👨‍💻 程式交易開發者操作流程

1. 下載 **MAS 中介系統（Bridge Service）**  
2. 登入平台帳號進行驗證與授權  
3. 使用 mas_lib 進行策略開發與回測  
4. 取得包含帳戶權益資訊的完整報表（含獲利、風控、KPI）  
5. 可選擇將策略上傳平台供個人使用或開放交易

## 🧩 產品內容架構

### 🔹 MAS Web 平台功能

註冊會員  
產生 API 金鑰  
加值付款與儲值 MAS Coin  
使用 AI 策略助理（生成策略、回測報表、視覺化圖表）  
下載策略專屬交易執行檔  

### 🔸 MAS Soft 本機端應用程式

安裝 mas-python-lib  
操作 MT5 自動化交易、策略測試與模擬  
輸出動態與靜態報表  
綁定帳號後取得真實權益與授權策略資料
